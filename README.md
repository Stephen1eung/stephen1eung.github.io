# Stephen Leung's Portfolio

A modern, minimalist portfolio website built with vanilla HTML, CSS, and JavaScript showcasing professional experience, projects, education, and technical skills.

## Project Structure

```
stephen1eung.github.io/
├── index.html          # Main HTML file
├── style.css           # Complete stylesheet with responsive design
├── script.js           # Interactive JavaScript functionality
├── assets/             # Static assets
│   ├── headshot.jpg    # Profile image
│   ├── about.jpg       # About section image
│   ├── cat.jpg         # My cat photo
│   ├── project-nba.jpg # NBA Analysis project thumbnail
│   └── project-xpired.jpg # Xpired project thumbnail
└── README.md           # This file
```

### Colors Used
- **Background**: Light neutral palette (`#f7f7f5`)
- **Surface**: White cards with subtle shadows
- **Text**: Dark gray (`#111827`) for high contrast
- **Muted**: Medium gray (`#4b5563`) for secondary text
- **Accent**: Blue (`#1f6feb`) for links and highlights

### Sections
1. **Home**: Hero section with call-to-action buttons and social links
2. **About**: Personal introduction with interests, hobbies, and technical stack
3. **Experience**: Interactive timeline with work history and skill tags
4. **Projects**: Grid layout showcasing personal projects
5. **Education**: Academic background and certifications
6. **My Cat**: Personal touch with fun facts
