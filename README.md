# Personal Portfolio Website

Welcome to my personal portfolio website! This is a modern, responsive website built with HTML, CSS, and JavaScript to showcase my skills, projects, and contact information.

## Features

- **Responsive Design**: Looks great on all devices
- **Modern UI**: Clean and professional design
- **Dark Mode**: Automatically adapts to system preferences
- **Smooth Scrolling**: For better navigation experience
- **Interactive Elements**: Hover effects and animations

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. That's it! No build process or dependencies required

## Customization

1. Update the content in `index.html` with your personal information
2. Replace the placeholder images in the projects section
3. Update the social media links in the contact section
4. Modify colors and styles in `css/styles.css`
5. Add custom JavaScript functionality in `js/main.js`

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter)
